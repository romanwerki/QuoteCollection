# Тест-план приложения "Коллекция цитат"

## Тест-кейсы

| № | Действие | Ожидаемый результат | Статус |
|---|----------|---------------------|--------|
| 1 | Запустить python main.py | Открывается главное окно 1100x700 | ✅ |
| 2 | Ввести текст + выбрать автора + нажать "Добавить" | Цитата появляется в таблице | ✅ |
| 3 | Попытаться добавить пустую цитату | QMessageBox "Введите текст цитаты" | ✅ |
| 4 | Установить будущую дату + "Добавить" | QMessageBox "Дата не может быть в будущем" | ✅ |
| 5 | Выбрать цитату + нажать "Изменить" | Данные сохраняются, таблица обновляется | ✅ |
| 6 | Выбрать цитату + нажать "Удалить" | Подтверждение → удаление из БД | ✅ |
| 7 | Выбрать цитату + нажать "Копировать" | Текст в буфере обмена | ✅ |
| 8 | Нажать "Загрузить фото" + выбрать PNG | Портрет 250x250 отображается | ✅ |
| 9 | Нажать "Очистить" | Форма очищается | ✅ |
| 10 | Закрыть окно (крестик) | Диалог подтверждения выхода | ✅ |
| 11 | Нажать Ctrl+N | Форма очищается | ✅ |
| 12 | Нажать Ctrl+D | Удаление выбранной цитаты | ✅ |
| 13 | Нажать Ctrl+E | Диалог сохранения JSON | ✅ |
| 14 | Нажать F5 | Таблица обновляется из БД | ✅ |
| 15 | Перезапустить приложение | Данные сохраняются в SQLite | ✅ |

## Граничные случаи

| Тест | Ожидаемое поведение |
|------|---------------------|
| Текст > 50 символов | В таблице обрезается с "..." |
| Изображение < 50px | QMessageBox "Изображение слишком маленькое" |
| Несуществующий файл портрета | QMessageBox "Файл не найден" |
| Повторный запуск | БД открывается корректно |
