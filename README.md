# Коллекция цитат

Приложение для управления коллекцией цитат с портретами авторов.

## Возможности
- Добавление, редактирование, удаление цитат
- Категоризация (философия, юмор, мотивация и др.)
- Избранные цитаты
- Портреты авторов
- Копирование в буфер обмена
- Валидация данных

## Технологии
- Python 3.12
- PyQt6
- SQLite3
- Pillow

## Запуск
1. Установите зависимости:
pip install -r requirements.txt
2. Запустите приложение:
python main.py

## Структура
- `main.py` — главный файл приложения
- `database.py` — работа с базой данных
- `ui_main.ui` — интерфейс (Qt Designer)
- `requirements.txt` — зависимости
