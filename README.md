# QuoteCollection — Коллекция цитат

Десктопное приложение для хранения, управления и каталогизации цитат известных людей. Разработано в рамках учебной практики с использованием PyQt6, SQLite и Pillow.

## Возможности

- Добавление, редактирование и удаление цитат
- Выбор автора и категории из предзаданных списков
- Отметка избранных цитат
- Загрузка и отображение портрета автора (Pillow)
- Копирование цитаты в буфер обмена
- Экспорт всех цитат в JSON
- Валидация ввода (проверка пустого текста, будущей даты)
- Сохранение размера окна между запусками
- Горячие клавиши: `Ctrl+N`, `Ctrl+D`, `Ctrl+E`, `F5`
- Логирование действий (INFO/WARNING/ERROR)

## Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/romanwerki/QuoteCollection.git
   cd QuoteCollection
   ```

2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   ```

3. Активируйте виртуальное окружение:
   - **Windows:** `venv\Scripts\activate`
   - **macOS/Linux:** `source venv/bin/activate`

4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

5. Запустите приложение:
   ```bash
   python main.py
   ```

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+N` | Очистить форму |
| `Ctrl+D` | Удалить выбранную цитату |
| `Ctrl+E` | Экспорт всех цитат в JSON |
| `F5` | Обновить таблицу из БД |

## Архитектура

Приложение построено по принципу разделения ответственности:

- **`main.py`** — UI-слой: интерфейс, сигналы/слоты, обработка событий, QSS-стилизация
- **`database.py`** — слой данных: подключение к SQLite, CRUD-операции, транзакции
- **`ui_main.ui`** — дизайн интерфейса, созданный в Qt Designer с использованием менеджеров компоновки (QHBoxLayout, QVBoxLayout)

Взаимодействие: UI отправляет запросы в `DatabaseManager`, который выполняет SQL-запросы с параметризацией (защита от SQL-инъекций).
