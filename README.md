# Коллекция цитат (PyQt5 Practice)

Приложение для управления коллекцией цитат с портретами авторов.

## Функциональность
- Добавление, редактирование, удаление цитат (CRUD)
- Категоризация цитат
- Избранные цитаты
- Портреты авторов (Pillow)
- Копирование цитат в буфер обмена
- Валидация данных
